# ガベージ コレクター関連の問題のデバッグ

ベーム・デマース・ヴァイザーの保守的なガベージ コレクター。どちらにも当てはまります

コレクター自体のバグが疑われる場合は、次のことを強くお勧めします。
続行する前に、最新のコレクター リリースを試してください。
